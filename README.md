# Digital Mycelium Public Showcase

**AI can generate faster than it can remember.**

Digital Mycelium v0.3 is a local deterministic source-return engine for AI-era artifact continuity.

It receives a file, ZIP, or folder; records structure; generates trace capsules and route receipts; exports a packet; re-ingests that packet; and verifies whether the route held.

The model does not decide HELD.  
The institution does not get the first word.  
The receipt carries the first account.
